# Учебные работы по программированию (ВТиИС 12-О, ЗКИиТУ)

Данный репозиторий предназначен для хранения моих учебных проектов и домашних заданий.
Здесь собраны решения задач, которые можно запустить для проверки.

## Содержание

- `card_check.py` — Скрипт для валидации номера банковской карты. Использует алгоритм Луна для проверки корректности ввода.

## Как запустить

1.  Убедитесь, что у вас установлен Python версии 3.x.
2.  Склонируйте репозиторий или скачайте файлы.
3.  Запустите интересующий скрипт через терминал:

```bash
python card_check.py
```
